In this R code file "StatResearchProject.r", I calculate the Kurtosis scores with 95% CI for 3 neuroscience studies. The Kurtosis scores are based on each study's respective dependent and independent variables. Group n's and variance ratios between treatment and control groups are also calculated. At the very end, a forest plot is made to visualize the Kurtosis score and CI. This is an ongoing project, so more studies will be added to the script.  
"StatResearchProject_main.r" contains the most up-to-date (as of 6/18/2026) code for the research project "Variance Heterogeneity and Heavy-Tailed Outcomes in Experimental Neuroscience Data". C ratios and Kurtosis estimates are calculated for the control and treatment groups within 30 interventional neuroscience studies. Group n's and variance ratios between treatment and control groups are also calculated. At the very end, a forest plot is made to visualize the Kurtosis score and CI. A meta-analysis of the kurtosis estimate has been added, along with a bootstrap resampling of the variance ratio to confirm significance.
